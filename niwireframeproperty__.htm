NiWireframeProperty

Свойства, делающие из объекта решетку из линий.
Аналог консольной команды TWF в игре.
В основном может использоваться в качестве "спец эффекта" - например для получения Голограмм Странных Двемерских.
 
И да, если сделать высоко полигональный объект, наложить на него "яркую" альфу (флаг 13 и пр) а затем добавить эти свойства,
то такой объект будет лучше выглядеть в удалении!
Т.е. сохраняется высокая плотность пикселей.
Что (хорошо) использовать во всякого рода взрывах.
См. robowind от уважаемого Shadow Mimicry.
 
An NiWireframeProperty allows applications to switch between drawing geometry as solid or as wireframe outlines.
NiWireframeProperty specifies how the polygons in NiGeometry objects are displayed. Supported modes are wireframe and solid (filled). Since many platforms experience performance degradations when wireframe mode is used, it is recommended only for debugging and tool purposes.

Создание в 3д МАХ.
Все модули.
Задается в свойствах материала.
 
Редактирование Nif.xml файла.
- Не требуется.
 
Примечание.
В отличие от niLines создаются автоматически для всех граней полигонов.
Как косых, так и прямых.
Т.е. практически невозможно управлять их рисунком.
 
Примечание.
Эти свойства, в игре, всегда активны для всех RootCollisionNode объектов.
Т.е. автоматически к ним применяются на уровне движка.

Примечание.
Эти свойства можно использовать для бегагинга скиненных объектов.
Т.е. в редакторе и в игре, будут видны грани полигонов и проблемные места на их стыке.
Что позволит залатать дыры.
Для просмотра анимаций поверхности, переключитесь на Preview Window и переключайтесь между группами анимаций.
 
Нифскоп умеет отображать эти свойства.
RootCollisionNode объекта.
Скриншот из редактора.
Хорошо видно наличие этих свойств.
 
Временно подключив эти свойства к файлу (существа например) можно удобно наблюдать поведение скининга.
А затем подправить оный в 3д МАХе.
 
Равно, если модель имеет "щели" их будет заметно в этом режиме отображения. В некоторых случаях, по крайней мере.
Как вариант - сначала продублировать базовую поверхность, к копии добавить эти свойства + zбуфер с флагом 1 и поднять яркость светимости материала.
 
Либо эти свойства можно добавить в качестве некоего "эффекта".
При использовании ЮВ контроллера текстур, они даже будет мерцать и менять свой цвет.
