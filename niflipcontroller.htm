NiFlipController

 
Контроллер анимации потока текстур.
 Т.е. работает как кинолента, прокручивает кадры с определенной скоростью и в определенной последовательности.
 
Может использоваться на любом из текстурных слотов.
Как по отдельности, так и на всех сразу.
За некоторыми исключениями, см. примечания.
 
The NiFlipController changes which image an NiTexturingProperty-Map will use.
It will cycle through its internal list of images according to a set of user-specified parameters.
